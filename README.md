# OpenClaw JSON Config Editor

OpenClaw JSON Config Editor 是一款单文件 HTML 应用程序，用于可视化编辑 JSON 配置文件。该工具将复杂的 JSON 结构转换为直观的表单界面，使用户能够轻松编辑配置而无需手动编写 JSON。

## 功能特色

- **三步流程**：导入 → 编辑 → 导出，顶部步骤条实时显示进度
- **智能表单识别**：布尔值 → 胶囊单选、枚举字符串 → 下拉选择、数组简单值 → 复选框（可动态增删）、嵌套对象 → 折叠卡片
- **输出三种模式**：格式化 / 压缩 / 语法高亮（关键字着色）
- **一键复制 & 下载**：可一键复制 JSON 到剪贴板或下载为 .json 文件
- **深色主题**：美观的深色界面，字体使用 Syne + DM Mono，需联网加载 Google Fonts（离线下仍可正常使用，字体会回退）

## 使用方法

1. **导入**：将您的 JSON 配置粘贴到输入区域，点击"解析并生成表单"
2. **编辑**：通过可视化表单编辑各项配置
3. **导出**：查看生成的 JSON，选择输出格式，复制或下载

## 技术特性

- 单文件 HTML，下载后直接双击浏览器打开即可，无需任何依赖或服务器
- 支持嵌套对象和数组的可视化编辑
- 智能识别常见配置项类型（如布尔值、枚举等）
- 实时统计字段数量和类型分布
- 语法高亮显示，便于阅读

## 支持的数据类型

- **字符串**：普通文本输入框
- **数字**：数字输入框
- **布尔值**：真/假单选按钮
- **枚举**：下拉选择框（支持自定义选项）
- **数组**：复选框组（简单值）或文本域（复杂值）
- **对象**：折叠卡片式嵌套表单

## 系统要求

- 现代浏览器（Chrome、Firefox、Edge、Safari）
- 无特殊硬件要求
- 无需网络连接（除首次加载Google Fonts外）

## 隐私说明

所有数据处理均在本地浏览器中进行，不会上传任何数据到服务器。
